# プレゼンテーション自動生成システム

> フォームに記入して、2つのファイルをClaudeに貼り付けるだけ。
> どのPCでも・誰でも・同じ品質のスライドが自動で完成します。

---

## 📁 ファイル構成

| ファイル | 役割 |
|---------|------|
| `CLAUDE_INSTRUCTION.md` | **設計書**（デザインルール・作り方が書いてある） |
| `presentation_form.txt` | **入力フォーム**（ここに内容を記入する） |
| `presentation_blueprint.pptx` | デザイン見本（参考用） |

---

## 🚀 使い方（3ステップ）

### STEP 1｜2つのファイルをダウンロード
GitHubから以下の2つをダウンロードしてください。

- `CLAUDE_INSTRUCTION.md`
- `presentation_form.txt`

---

### STEP 2｜フォームに内容を記入
`presentation_form.txt` をテキストエディタで開き、
`〇〇` の部分を自分の内容に書き換えてください。

```
会社名: 〇〇株式会社　　→　会社名: 株式会社プレゼンの達人
サービス名: 〇〇　　　　→　サービス名: プレゼン上達の為の講義
```

---

### STEP 3｜Claudeに送るだけ

**↓↓↓ 以下をそのままコピーして、2つのファイルの内容とともにClaudeに送ってください ↓↓↓**

```
━━━ここから貼り付け━━━

【設計書】
（CLAUDE_INSTRUCTION.md の内容を全部貼り付け）

【フォーム】
（記入済みの presentation_form.txt の内容を全部貼り付け）

上記の設計書とフォームをもとに、プレゼンテーションを自動生成してください。

━━━ここまで━━━
```

---

### STEP 4｜完成！
Claudeが自動でPPTXファイルを生成・保存します。

## ❓ よくある質問

**Q: 特定のスライドだけ内容を変えたい**
→ Claudeに「〇〇スライドの〇〇を変更して」と伝えてください。

**Q: スライドを追加したい**
→ Claudeに「〇〇のスライドを追加して」と伝えてください。

**Q: デザインを変えたい**
→ 管理者（岡本）に相談してください。社内共通デザインのため。
