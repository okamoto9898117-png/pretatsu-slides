# プレゼンテーション自動生成システム v4.0

> **どの社員が指示しても、世界水準の同じクオリティのスライドが自動生成されます。**
> フォームに記入して、2つのファイルをClaudeに貼り付けるだけ。

---

## 📁 ファイル構成

| ファイル | 役割 |
|---------|------|
| `README.md` | **この使い方ガイド** |
| `CLAUDE_INSTRUCTION.md` | **設計書**（Claudeへの指示・デザインルール全部入り） |
| `presentation_form.txt` | **入力フォーム**（内容を記入するだけ） |
| `pretatsu_editorial_v5.pptx` | Editorial Style 最高品質サンプル |
| `pretatsu_v4_final.pptx` | カラキャス参考 標準スタイルサンプル |

---

## 🚀 使い方（3ステップ）

### STEP 1｜2つのファイルをダウンロード
GitHubから以下の2つをダウンロード：
- `CLAUDE_INSTRUCTION.md`（設計書）
- `presentation_form.txt`（入力フォーム）

---

### STEP 2｜フォームに内容を記入
`presentation_form.txt` をメモ帳などで開き、`〇〇` を書き換える。

```
会社名: 〇〇株式会社　→　会社名: 株式会社プレゼンの達人
サービス名: 〇〇　　　→　サービス名: プレゼン講座
テーマ: Editorial（写真×大型タイポ）　← これだけで超高品質に
```

---

### STEP 3｜Claudeに送るだけ

**↓↓↓ 以下をそのままコピーして2ファイルの内容と一緒にClaudeへ ↓↓↓**

```
━━━ここから貼り付け━━━

【設計書】
（CLAUDE_INSTRUCTION.md の内容を全部貼り付け）

【フォーム】
（記入済みの presentation_form.txt の内容を全部貼り付け）

上記の設計書とフォームをもとに、プレゼンテーションを自動生成してください。

━━━ここまで━━━
```

## ❓ よくある質問

**Q: 特定のスライドだけ変えたい**
→ Claudeに「〇〇スライドの〇〇を変更して」と伝えてください。

**Q: 自社ブランドカラーを使いたい**
→ フォームの「テーマ: カスタム」でカラーコードを指定してください。

**Q: もっとスライドを追加したい**
→ Claudeに「〇〇のスライドを追加して」と伝えてください。

**Q: デザインの基準を変えたい**
→ 管理者（岡本）に相談。`CLAUDE_INSTRUCTION.md` を更新します。
