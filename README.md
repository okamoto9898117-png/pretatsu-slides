# プレゼンテーション自動生成システム

> 入力フォームを記入してClaudeに貼り付けるだけで、プロ品質のスライドが自動生成されます。

---

## 📁 ファイル構成

| ファイル | 用途 |
|---------|------|
| `presentation_form.txt` | **入力フォーム**（ここに内容を記入する） |
| `generate_from_form.js` | 自動生成スクリプト（触らなくてOK） |
| `presentation_blueprint.pptx` | デザイン見本（ベーステンプレート） |

---

## 🚀 使い方（3ステップ）

### STEP 1｜フォームに内容を記入する
`presentation_form.txt` をテキストエディタで開いて
「〇〇」の部分を自分の内容に書き換えてください。

```
会社名: 株式会社〇〇　→　会社名: 株式会社プレゼンの達人
サービス名: 〇〇　　　→　サービス名: プレゼン上達の為の講義
...
```

### STEP 2｜Claudeに貼り付ける
記入したフォームの内容を**全部コピー**して、
Claudeに以下のメッセージとともに貼り付けてください：

```
このフォームからプレゼンを作成してください

（フォームの内容をここに貼り付け）
```

### STEP 3｜完成！
Claudeが自動でPPTXファイルを生成して保存します。
`/Users/koudai/My Campany/` フォルダに保存されます。

## ❓ よくある質問

**Q: フォームの項目が全部埋まらない場合は？**
A: 空欄のまま「〇〇」にしておいてください。後で修正できます。

**Q: デザインを変えたい場合は？**
A: Claudeに「〇〇のデザインを変更して」と伝えてください。

**Q: スライドを追加・削除したい場合は？**
A: Claudeに「〇〇のスライドを追加して」と伝えてください。
