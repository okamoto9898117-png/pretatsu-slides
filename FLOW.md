## 📝 ① フォームに内容を記入

`presentation_form.txt` をメモ帳で開いて `〇〇` を書き換えるだけ。

```
【書き換える項目】

テーマ      → Editorial（推奨）/ プレミアム / スタートアップ など
会社名      → 株式会社プレゼンの達人
サービス名  → プレゼン講座
キャッチコピー → 将来の夢なんて今叶えろ
課題①②③   → 具体的な課題を記入
強み①②③   → サービスの特徴を記入
実績の数字  → 98%・+30% など
料金        → ¥〇〇〇,〇〇〇〜
連絡先      → TEL・EMAIL・WEB
```
