## 📐 設計原則（全テーマ共通・必ず守ること）

### ① 1スライド1メッセージ
- 1枚で伝えることは**1つだけ**
- タイトルだけ読めば内容がわかる状態にする

### ② KISS法則（Keep it Short and Simple）
- **体言止め**を使う（「〜する」→「〜の実現」）
- 箇条書きは3〜5項目まで
- 不要な装飾・重複テキストは削除

### ③ 配色の黄金比率
- ベースカラー：**70%**
- メインカラー：**25%**
- アクセントカラー：**5%**
- 色数は**最大3色**まで

### ④ 数字の見せ方（世界最高水準）
- 重要な数字は**32〜180pt**で巨大に表示
- 数字の横に単位（%・円・社）を添える
- 背景はダーク、数字はホワイト or ゴールド

### ⑤ 写真の使い方（Editorialスタイル）
- 写真はフルブリード（スライド全面）で使用
- 写真の上に黒透明オーバーレイを重ねてテキストを読みやすく
- `sizing: { type: "cover", w: 10, h: 5.625 }` を必ず使用

### ⑥ 余白の贅沢な使い方
- 最小マージン：0.5"以上
- 詰め込まない・空白を恐れない
- 余白こそが「高級感」を生む

## ⚡ Claudeへの実行指示

フォームの内容を読み取り、以下を実行すること：

1. **テーマを確認** → カラーパレットを決定
2. **Editorialテーマの場合** → Unsplash写真をhttpsでダウンロードして使用
3. **スクリプトを作成** → `/tmp/pptx_build/` でPptxGenJS Node.jsスクリプトを生成
4. **実行** → `node スクリプト名.js`
5. **保存先** → `/Users/[ユーザー名]/My Campany/presentation_[会社名].pptx`
6. **GitHubにアップロード** → `okamoto9898117-png/presentation-blueprint` リポジトリ

**使用ライブラリ：**
- `pptxgenjs` → PPTXファイル生成
- `react-icons/fa` → アイコン（FAアイコン）
- `sharp` → SVGをPNGに変換
- `https` → 写真ダウンロード（Node.js標準）

**注意事項：**
- 色コードに`#`を付けない（PptxGenJS仕様）
- shadowオブジェクトは使い回し禁止（毎回`makeShadow()`で生成）
- テキストの`margin: 0`を使って位置を正確に合わせる

## 🖼️ サムネイル自動生成（thumbnail_form.txt）

`thumbnail_form.txt` が貼り付けられたら、以下の手順でPNG画像を自動生成する。

### 実行手順

1. フォームを解析して以下のJSONを組み立てる
```json
{
  "mainTitle": "（メインタイトルの値）",
  "subTitle": "（サブタイトルの値、省略時は空文字）",
  "category": "（カテゴリの値、省略時は空文字）",
  "brand": "（ブランド名の値、省略時は空文字）",
  "theme": "（テーマの値）",
  "aspectRatio": "（アスペクト比の値）",
  "bgColor": "（カスタム背景カラー、カスタム時のみ）",
  "accentColor": "（カスタムアクセントカラー、カスタム時のみ）",
  "textColor": "（カスタムテキストカラー、カスタム時のみ）"
}
```

2. JSONを `/tmp/thumbnail_input.json` に保存する

3. 以下のコマンドを実行する：
```bash
node /tmp/pptx_build/create_thumbnail.js /tmp/thumbnail_input.json [出力パス]
```
- 出力パスはフォームの「ファイル名」と「保存先」から決定する
- 省略時は `/Users/koudai/My Campany/thumbnail.png`

4. 生成完了後、ファイルパスをユーザーに伝える

### テーマ → aspectRatio 対応表
| フォームの記載 | JSONの値 |
|---|---|
| 16:9（YouTube・プレゼン・一般用） | `16:9` |
| 1:1（SNS正方形） | `1:1` |
| youtube（YouTube専用） | `youtube` |
| ogp（ブログOGP） | `ogp` |
| 9:16（縦型） | `9:16` |
